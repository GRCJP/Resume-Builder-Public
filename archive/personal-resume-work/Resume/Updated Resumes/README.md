## Workflow

### **When to Update These:**
1. ‚úÖ New certification earned
2. ‚úÖ Major project completed
3. ‚úÖ New achievement with metrics
4. ‚úÖ Job change or promotion
5. ‚úÖ Quarterly review and refresh

### **How to Update:**
1. Open the master version
2. Add new achievement/certification
3. Update metrics if changed
4. Save with date: `Jonathan_Perez_GRC_Master_2025-11.docx`
5. Keep previous version as backup

### **Version Control:**
- Keep current version: `Jonathan_Perez_GRC_Master.docx`
- Keep previous version: `Jonathan_Perez_GRC_Master_2025-10.docx`
- Delete versions older than 3 months
