## Editing Tips

### **When Editing, Focus On:**

1. **Outcome-Focused Bullets**
   - ‚ùå "Responsible for vulnerability management"
   - ‚úÖ "Reduced vulnerabilities by 83% across 10,000+ assets"

2. **Top 3 Wins First**
   - Lead with your biggest achievements
   - Put them in first half of page 1
   - Make them impossible to miss

3. **Keywords from Job Description**
   - Copy exact terms from job posting
   - Integrate naturally into bullets
   - Don't keyword stuff

4. **Metrics Everywhere**
   - Add numbers to every bullet
   - Show scope (10,000+ assets)
   - Show impact (70% reduction)

5. **Tools and Frameworks**
   - Mention specific tools (AWS Config, Wiz, Qualys)
   - Name frameworks (NIST RMF, SOC 2, ISO 27001)
   - Show technical depth

## Troubleshooting

### **"I uploaded but can't edit"**
- Click the resume card to select it
- Then click "Edit" button
- Editor will open

### **"Changes didn't save"**
- Make sure you clicked "Save"
- Check that resume name isn't empty
- Verify content isn't empty

### **"Can't create new resume"**
- Check if you have 5 resumes already
- Delete one if at maximum
- Then try creating again

### **"Match score seems wrong"**
- Verify correct resume is selected
- Check that you saved recent edits
- Make sure job description pasted fully
